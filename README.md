# Simulação de Phishing – Projeto Educacional

Este projeto demonstra como ataques de **phishing** são estruturados e como podem ser analisados em ambiente seguro.  
A simulação permite estudar:

- Gatilhos de engenharia social  
- Rastreamento de cliques  
- Coleta básica de informações  
- Comportamento da vítima em cenários simulados  

---

##  Aviso Ético

Este projeto é estritamente para **uso educacional**.  
Não deve ser utilizado para atacar, prejudicar ou coletar dados reais de terceiros.

Toda experimentação deve ser feita **somente em ambientes controlados**, como laboratórios pessoais.

---

##  Objetivo

Criar uma simulação realista para estudo de engenharia social, incluindo:

- Página falsa responsiva  
- Coleta de user-agent  
- Registro de cliques e eventos  
- Captura controlada de credenciais (didática)  
- Análise posterior via logs  

O foco é **conscientização e aprendizado**, nunca abuso.

---

##  Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Python (servidor e logs)  
- SMTP (opcional, para simulação de envio)
